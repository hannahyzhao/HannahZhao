- Developed an Android application that uses computer vision to infer user emotional state from facial input and translate it into actionable behavioral recommendations.
- Implemented a real-time emotion recognition pipeline using OpenCV and TensorFlow models to process image data and classify facial expressions.
- Designed a feedback system that maps detected emotional states to personalized activity suggestions (e.g., music, mindfulness, physical activity), creating a closed-loop interaction between perception and action.
- Built mobile application architecture in Java with Firebase integration for data storage and user state persistence.
- Focused on human-centered AI design, exploring how machine perception can influence user behavior and well-being.
